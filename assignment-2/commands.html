Git Basic Commands

Configuring user information:

To set the user-name:
$ git config --global user.name "username"

To set the user-email:
$ git config --global user.email "useremail"

To enable colors of the command line input:
$ git config --global color.ui auto

Initializing and cloning repositories:

To initialize existing directory:
$ git init

To retrieve an entire repository from a hosted location via URL/SSH:
$ git clone [URL/SSH]

Snapshots and staging area:

To show modified files, staged for next commit:
$ git status

To add a file:
$ git add [file name]

To unstage a file:
$ git reset [file name]

To show the difference what is changed but not staged:
$ git diff

To show the difference what is staged but not committed:
$ git diff --staged

To commit your staged files:
$ git commit -m "your message"

Working with branches:

To create a new branch:
$ git branch [branch name]

To list branches. a* will appear next to the current active branch:
$ git branch

Switch to another branch and check it out into working directory:
$ git checkout

To merge specified branch's history into the current one:
$ git merge [branch name]

To show all commits:
$ git log

Sharing and Updating Projects:

To push changes to remote repository:
$ git push

To push a branch to remote repository:
$ git push origin [branch name]

To delete a remote branch:
$ git push origin --delete [branch name]

To update local repository to the latest commit:
$ git pull

To pull changes from remote repository:
$ git pull origin [branch name]
